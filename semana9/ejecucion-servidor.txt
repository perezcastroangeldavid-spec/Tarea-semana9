Cuando ejecuto el comando node server.js, Node.js inicia la aplicación y el servidor comienza a escuchar solicitudes en el puerto 3000.

En la terminal aparece el mensaje:

Servidor ejecutándose en puerto 3000

Al visitar la ruta /estado, el servidor responde con un objeto JSON que indica que el servicio está funcionando correctamente:

{
    "estado": "Servidor funcionando",
    "servicio": "API comunitaria"
}
}
